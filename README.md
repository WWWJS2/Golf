# MAJESTY Golf Website - Flat GitHub Version

هذه نسخة مهيأة للرفع على GitHub Pages بدون أي مجلدات داخلية.

## طريقة الرفع
1. افتح مستودع GitHub.
2. اختر Add file > Upload files.
3. ارفع كل الملفات الموجودة هنا كما هي في الصفحة الرئيسية للمستودع.
4. تأكد أن ملف `index.html` موجود في الجذر الرئيسي للـ repository.
5. من Settings > Pages اختر Branch: main و Folder: /root ثم Save.

## ملاحظة
لا ترفع ملف ZIP نفسه. فك الضغط أولًا، ثم ارفع الملفات كلها مرة واحدة.
